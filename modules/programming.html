Программирование Arduino

На главную

Основы языка Arduino (C++)

Arduino использует язык программирования, основанный на C++. Каждая программа должна содержать как минимум две функции:

void setup() {
    // Код здесь выполняется один раз при запуске
}

void loop() {
    // Код здесь выполняется циклически
}
Совет: setup() используется для инициализации, а loop() для основной логики программы.

Переменные и типы данных

Основные типы данных в Arduino:

// Целые числа
int number = 42;           // -32768 до 32767
long bigNumber = 100000;   // большие числа

// Дробные числа
float temperature = 25.5;  // числа с плавающей точкой

// Логический тип
boolean isOn = true;      // true или false

// Символы
char letter = 'A';        // одиночный символ

// Константы
const int LED_PIN = 13;   // значение нельзя изменить

Условные операторы

Управление потоком программы:

// if-else
if (temperature > 25) {
    digitalWrite(FAN_PIN, HIGH);
} else {
    digitalWrite(FAN_PIN, LOW);
}

// switch-case
switch (sensorValue) {
    case 1:
        // действие 1
        break;
    case 2:
        // действие 2
        break;
    default:
        // действие по умолчанию
}

Циклы

Различные типы циклов:

// for цикл
for (int i = 0; i < 5; i++) {
    digitalWrite(LED_PIN, HIGH);
    delay(1000);
    digitalWrite(LED_PIN, LOW);
    delay(1000);
}

// while цикл
while (digitalRead(BUTTON_PIN) == HIGH) {
    // выполнять, пока кнопка нажата
}

// do-while цикл
do {
    // выполнится хотя бы раз
} while (condition);

Функции

Создание собственных функций:

// Функция без возвращаемого значения
void blinkLED(int times) {
    for (int i = 0; i < times; i++) {
        digitalWrite(LED_PIN, HIGH);
        delay(1000);
        digitalWrite(LED_PIN, LOW);
        delay(1000);
    }
}

// Функция с возвращаемым значением
float readTemperature() {
    int sensorValue = analogRead(TEMP_PIN);
    float temperature = (sensorValue * 5.0 / 1024.0) * 100;
    return temperature;
}

Массивы

Работа с массивами данных:

// Объявление массива
int ledPins[] = {2, 3, 4, 5, 6};

// Перебор массива
for (int i = 0; i < 5; i++) {
    pinMode(ledPins[i], OUTPUT);
}

// Двумерный массив
int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
