Аналоговые входы Arduino

Arduino имеет несколько аналоговых входов (A0-A5), которые могут измерять напряжение в диапазоне от 0 до 5В с разрешением 10 бит (0-1023). Это позволяет считывать данные с аналоговых датчиков, таких как потенциометры, фоторезисторы и датчики температуры.

Чтение аналоговых значений

Для чтения аналоговых значений используется функция analogRead()

void setup() {
    Serial.begin(9600);  // Инициализация последовательного порта
}

void loop() {
    int sensorValue = analogRead(A0);  // Чтение значения с аналогового входа A0
    float voltage = sensorValue * (5.0 / 1023.0);  // Преобразование в напряжение
    Serial.println(voltage);  // Вывод значения
    delay(100);
}

ШИМ (PWM) выходы

Arduino может генерировать аналогоподобные сигналы с помощью ШИМ (широтно-импульсной модуляции) на специальных пинах, помеченных знаком ~. ШИМ позволяет управлять яркостью светодиодов, скоростью двигателей и другими аналоговыми устройствами.

void setup() {
    pinMode(9, OUTPUT);  // Настройка ШИМ пина
}

void loop() {
    // Плавное увеличение яркости
    for(int i = 0; i <= 255; i++) {
        analogWrite(9, i);
        delay(10);
    }
    
    // Плавное уменьшение яркости
    for(int i = 255; i >= 0; i--) {
        analogWrite(9, i);
        delay(10);
    }
}

Работа с потенциометром

Потенциометр - это переменный резистор, который часто используется для создания регулируемого напряжения.

const int potPin = A0;    // Пин потенциометра
const int ledPin = 9;     // ШИМ пин для светодиода

void setup() {
    pinMode(ledPin, OUTPUT);
}

void loop() {
    int potValue = analogRead(potPin);
    int brightness = map(potValue, 0, 1023, 0, 255);
    analogWrite(ledPin, brightness);
}
