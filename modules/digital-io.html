Цифровые пины Arduino

Цифровые пины Arduino могут находиться в одном из двух состояний: HIGH (высокий уровень, 5В) или LOW (низкий уровень, 0В). Они используются для подключения различных устройств, таких как светодиоды, кнопки, реле и другие цифровые компоненты.

Подключение и управление светодиодом

Самый простой пример использования цифрового выхода - это управление светодиодом.

void setup() {
    pinMode(13, OUTPUT);  // Настраиваем пин 13 как выход
}

void loop() {
    digitalWrite(13, HIGH);  // Включаем светодиод
    delay(1000);            // Ждем 1 секунду
    digitalWrite(13, LOW);   // Выключаем светодиод
    delay(1000);            // Ждем 1 секунду
}

Работа с кнопками

Кнопки используются как цифровые входы. При подключении кнопки важно использовать подтягивающий резистор.

const int buttonPin = 2;  // Пин кнопки
const int ledPin = 13;    // Пин светодиода

void setup() {
    pinMode(buttonPin, INPUT);
    pinMode(ledPin, OUTPUT);
}

void loop() {
    if (digitalRead(buttonPin) == HIGH) {
        digitalWrite(ledPin, HIGH);
    } else {
        digitalWrite(ledPin, LOW);
    }
}
